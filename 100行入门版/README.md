windows 可解压 voicegame.zip 后，直接运行 game.exe
确保电脑上接有麦克风或其他声音输入设备。请勿解压在有中文路径的目录下。

Mac 和 Linux 需通过源码运行

```
python game.py
```

代码基于 Python 2，运行前需安装 cocos2d 和 pyaudio 模块

```
pip install cocos2d
pip install pyaudio
```
